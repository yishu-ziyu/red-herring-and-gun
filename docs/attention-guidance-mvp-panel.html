结论（注意力引导后）

主管部门已公开否认存在「取消退休金」政策新华社 ↗人社部回应 ↗。 网传「内部文件」目前无法追溯到可验证原始出处缺口 · P0。 多条自媒体转载内容经谱系折叠后属于同一上游转载折叠 · 非独立， 不能当作多源共识。 因此当前不能推出不可说「政策已确定取消」。

Can say

  • 官方已否认该政策文件存在 新华社 ↗
  • 公开渠道未见可验证原件 缺口

Cannot say

  • 不能说「政策已落地取消」 blocked
  • 不能把转载热度当成多源证实 blocked

先看这 3 处

Attention Rail · 按风险排序，不是证据全量列表

P0
反证：官方否认 与原说法直接冲突。先点开主管部门回应，再决定是否转发。
P1
假多数：转载已折叠 5 条链接 → 1 个上游。界面应显示「非独立」，避免被数量欺骗。
P2
缺口：原件不可追溯 你还需要：可验证的文件扫描件 / 官方发布号 / 原始链接。
  1. A 结论正文改为 ClaimSpan 序列
  2. B 行内来源芯片，点击开抽屉
  3. C 右侧 Attention Rail，只推 3 处
  4. D can/cannot say 挂 license 标记

Layer 1 · 句级溯源

assert 后挂来源芯片；gap / blocked 不允许装成已证实。

Layer 2 · 注意力优先级

P0 反证冲突 → P1 假多数 → P2 缺口。背景信息默认退后。

Layer 3 · 缺口发现

明确「你还需要什么」，并接到递归搜索入口，但不自动连跑。
